#How to Install ios Apllication Usig Flutter

Open Terminal in Mac

Enter Folowing Code Below :

1)Source ~/.bashrc
2)cd "Project Path"
3)flutter clean
4)flutter pub get
5)flutter pub upgrade
6)cd ios
7)pod install
8)cd ..
9)flutter build ios
10) Open Xcode and Run Project in Your Divice
